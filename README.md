# 🚀 My Prompt Engineering & AI Lab

Welcome to my repository! This is where I document my journey learning **Prompt Engineering**, mastering advanced LLM frameworks, and building lightweight interactive applications.

## 🧠 Core Frameworks I Use

I utilize structured prompt frameworks to control AI behaviors, logic, and output formats:
* **Role / Persona:** Anchor tone and technical depth.
* **Context & Constraints:** Enforce strict boundaries (e.g., specific word counts, avoiding fluff).
* **Delimiters:** Isolating input data using markdown syntax (`"""` or `<tags>`).
* **Chain-of-Thought (CoT):** Forcing step-by-step logical reasoning for complex problems.
